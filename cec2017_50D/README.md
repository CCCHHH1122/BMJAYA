# CEC-2017 50D Validation

This folder contains the code and results for the high-dimensional CEC-2017 validation experiment.

Run from MATLAB:

```matlab
cd('cec2017_50D/code')
run_cec2017_50D_all
```

The experiment uses `D = 50`, population size `N = 50`, `30` independent runs, and `MaxFEs = 500000`. Function F2 is excluded following the instability note in the manuscript.
